✉️ Registration Confirmation Emails

When someone registers for an event, send them an automatic confirmation email from noreply@kiitfilmsociety.in. Requires an SMTP relay that allows sending from your domain (e.g. Zoho Mail, Google Workspace, Brevo, Postmark).

Get this from Brevo → Settings → SMTP & API → API Keys

The email is sent only when the registration form has an Email-type question. Date & venue lines are auto-populated from the event details.
